Wiki directories

  1. ./subsubwiki1/ : Highlighted as missing-link (?) (non-reduced path)
  2. subsubwiki2/ : Highlighted as link
  3. ../subwiki2/ : Highlighted as link
  4. [[..\subwiki2/]] : nonstandard path too highlighted as link
  5. [[..\subwiki2\]] : nonstandard path too highlighted as link
  6. ../wiki/ : Highlighted as missing-link
  7. ../../wiki/ : Highlighted as missing-link (the path leaves the wiki)
  8. ./ : Highlighted as link
  9. a/ : Highlighted as missing-link
  10. ../ : Highlighted as link
  11. [[..\]] : nonstandard path too highlighted as link
  12. ../../ : Highlighted as missing-link

Wikilinks To Non-Wiki Local Resources

  1. Path/To/Nowhere.txt : Highlighted as missing-link. Schemeless URLs are equivalent to wiki-scheme URLs, which are fixed with an .html extension
  2. file:///Absolute/Path/To/Nowhere.txt : Always highlight absolute paths as links. No .html extension added
